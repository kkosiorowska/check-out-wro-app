# Check Out Wro

## Description
Tourist application to check out places in Wroc≈Çaw. User has posibility to look for attractions. After click marker on map or item from list user will see more information about place.
* [Demo Version](https://react-check-wro.firebaseapp.com/)

### Features
* Http Requests
* React Router
* Redux
* Firebase
* Leaflet Map
